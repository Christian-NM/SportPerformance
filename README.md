# Sport Performance 🏋️‍♂️

**Sport Performance** es una aplicación de escritorio moderna desarrollada en C# y WPF para la gestión avanzada de entrenamientos de fuerza. Permite a los usuarios llevar un control estricto de sus ejercicios, calcular su RM (Repetición Máxima) de forma automática y gestionar su progreso con una interfaz intuitiva y fluida.

## 🚀 Características Principales

* **Arquitectura MVVM**: Separación limpia entre la lógica de negocio y la interfaz de usuario.
* **Base de Datos Local**: Persistencia de datos robusta mediante SQLite.
* **Buscador Inteligente**: Filtrado de ejercicios en tiempo real por nombre o grupo muscular.
* **UX/UI Premium**: Diseño basado en tarjetas (Cards), sombras de alta fidelidad y navegación animada.
* **Formateo Automático**: Sistema inteligente que estandariza los nombres de los ejercicios (Capitalización de títulos).
* **Cálculo de RM**: Algoritmo integrado para estimar la fuerza máxima teórica.

## 🛠️ Stack Tecnológico

* **Lenguaje:** C#
* **Framework:** .NET (WPF)
* **Base de Datos:** SQLite
* **Patrón de Diseño:** MVVM (Model-View-ViewModel)
